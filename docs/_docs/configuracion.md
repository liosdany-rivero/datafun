# Configuración del sistema

Guía para configurar DataFun ERP según las necesidades de tu empresa.

## Configuración básica

### Datos de la empresa
- **Información general**: Nombre, RIF, dirección, teléfono
- **Logo empresarial**: Sube el logo de tu empresa
- **Información fiscal**: Datos para facturación

### Parámetros del sistema
- **Moneda principal**: Bolívares, Dólares, etc.
- **Formato de fecha**: DD/MM/AAAA, MM/DD/AAAA
- **Zona horaria**: Configuración regional

## Configuración de usuarios

### Crear usuarios
1. Ve a **Configuración → Usuarios**
2. Haz clic en **Nuevo Usuario**
3. Completa: nombre, email, contraseña
4. Asigna permisos correspondientes

### Roles y permisos
- **Administrador**: Acceso completo al sistema
- **Gerente**: Acceso a reportes y gestión
- **Vendedor**: Acceso limitado a ventas
- **Almacén**: Solo gestión de inventario

## Personalización

### Facturas y documentos
- Numeración automática
- Formato personalizado
- Textos personalizados

### Notificaciones
- Alertas por email
- Notificaciones en sistema
- Recordatorios automáticos

# Guía de acceso

DataFun ERP es una aplicación web, solo necesitas un navegador para usarla.

## Requisitos del sistema

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet
- Dispositivo (PC, laptop, tablet o smartphone)

## Primer acceso

1. **Abrir la aplicación**
   - Ve a la URL proporcionada por tu administrador
   - Ejemplo: `https://app.datafun.com`

2. **Iniciar sesión**
   - Usa las credenciales proporcionadas
   - Usuario administrador: `admin`
   - Contraseña: `[configurada por el administrador]`

## Configuración inicial

Después de iniciar sesión por primera vez:

1. **Completar datos de empresa**
   - Ve a Configuración → Empresa
   - Ingresa nombre, RIF, dirección, teléfono

2. **Personalizar preferencias**
   - Establece moneda local
   - Configura formatos de fecha y número
   - Define impuestos aplicables

## Soporte técnico

Si encuentras problemas de acceso:
- Verifica tu conexión a Internet
- Asegúrate de usar un navegador compatible
- Contacta al administrador del sistema

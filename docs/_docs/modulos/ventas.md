# Módulo de Ventas

Gestión completa del ciclo comercial de tu empresa.

## Gestión de clientes

### Agregar clientes
1. Ve a **Ventas → Clientes**
2. Haz clic en **Nuevo Cliente**
3. Completa: nombre, RIF, dirección, teléfono
4. Configura límite de crédito si aplica

### Categorías de clientes
- Clientes mayoristas
- Clientes minoristas
- Clientes especiales

## Proceso de venta

### Crear cotización
1. **Ventas → Nueva Cotización**
2. Selecciona cliente
3. Agrega productos/servicios
4. Aplica descuentos si corresponde

### Convertir a pedido
- Las cotizaciones aprobadas se convierten en pedidos
- Seguimiento de estado del pedido
- Notificaciones al cliente

### Facturación
- Generación automática de facturas
- Control de inventario automático
- Registro contable integrado

## Reportes de ventas

- Ventas por período
- Productos más vendidos
- Comisiones de vendedores
- Clientes con mayor facturación

# Módulo de Contabilidad

Sistema financiero integrado para el control completo de tu empresa.

## Plan de cuentas

### Estructura contable
- Activos
- Pasivos
- Patrimonio
- Ingresos
- Gastos

### Personalización
- Agregar cuentas personalizadas
- Configurar subcuentas
- Definir centros de costo

## Asientos contables

### Automáticos
- Ventas generan asientos automáticos
- Compras registran contablemente
- Nómina integrada

### Manuales
1. **Contabilidad → Asientos Manuales**
2. Selecciona fecha y descripción
3. Agrega débitos y créditos
4. Verifica balance (debe = haber)

## Reportes financieros

### Básicos
- Balance de comprobación
- Estado de resultados
- Balance general

### Avanzados
- Flujo de efectivo
- Análisis de ratios
- Comparativos por período

## Períodos fiscales

### Cierre mensual
- Conciliación bancaria
- Ajustes de cierre
- Reportes mensuales

### Cierre anual
- Cierre de ejercicio
- Apertura nuevo período
- Reportes anuales

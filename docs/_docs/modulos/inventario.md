# Módulo de Inventario

Control completo de productos, stock y almacenes.

## Gestión de productos

### Agregar productos
1. Ve a **Inventario → Productos**
2. Haz clic en **Nuevo Producto**
3. Completa: código, nombre, descripción, precio
4. Configura: categoría, impuestos, stock mínimo

### Categorías
- Organiza productos por categorías
- Jerarquía ilimitada
- Reportes por categoría

### Precios y costos
- Precio de venta
- Costo de compra
- Márgenes automáticos
- Promociones temporales

## Control de stock

### Movimientos de inventario
- **Entradas**: Compras, devoluciones
- **Salidas**: Ventas, gastos
- **Ajustes**: Correcciones de inventario

### Alertas de stock
- Stock mínimo
- Productos agotados
- Exceso de inventario

## Múltiples almacenes

### Configurar almacenes
- Almacén principal
- Puntos de venta
- Bodegas externas

### Traslados
- Entre almacenes
- Control de existencias
- Historial de movimientos

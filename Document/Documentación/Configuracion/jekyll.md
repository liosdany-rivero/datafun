# jekyll

Se Utiliza para visualizar la documentacion en un servidor localmente


## Crear el servidor
1. Navegar a tu carpeta docs
- cd G:\XAMPP\htdocs\datafun\docs
2. Instalar dependencias
- bundle install
3. Ejecutar servidor local
- bundle exec jekyll serve --livereload



## Iniciar el servidor
